# Stream-monitor
This project monitors streaming on Chzzk, Twitcast, X(space).

# Tutorial
1. Download

        git clone https://github.com/stream-utiles/stream-monitor.git

2. Install

        cd stream-monitor
        pip3 install -r requirements.txt

3. Configuration

        ./setting        

4. option


# Environment
- Raspberry PI 3B+
- 2.5' HDD 120GB (connected without extra power)
